Technical Blog Part 1

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML is the contents of a webpage, if you thought of a book or magazine then the html would be the unformatted words and photos.
The DOM interprets the HTML and generates elements.
CSS is the styling rules, this takes the HTML elements and formats the words and pictures. If you think about a magazine the CSS would be the part the lays everything out and makes it look good.

What is meant by boxifying design?

Boxifying a design is the process of taking a design of a webpage and splitting it up into box elements. These boxes would then become elements in your page design.

What is the box model?

The box model is the rule that lays out every element on a web page. Every element (text box, image, shape) is made of a box/boxes.
The box model is the nested layout around every box. This consists of content at the centre, padding around this then border then the margin surrounding the whole lot.
