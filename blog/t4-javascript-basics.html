How does JavaScript compare to HTML and CSS?

JavaScript it’s the brains that works behind the scenes on web pages. HTML gives the content along with some description to the content type, CSS takes the HTML and styles is based on the type descriptions. JavaScript works in the background to create the functionality of page.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control loops let a program repeat a task for a set amount of cycles or until a condition is met. For example putting on socks this would involve putting a sock on two times, we know we need to do this twice so we could use a for loop. Another example would be cleaning a plate we want to keep scrubbing until it is clean, we don’t know at the start how dirty the plate is so we could use a while loop.

Explain the difference between accessing data from arrays and object literals.

Arrays and object literals are quite similar they are both used to store multiple values nested within a parent component. Arrays are stored with a numeric index, so to access a specific value you must know the exact location of the value. Object literals store values with a string description index, this means that the value can be accessed with just a description.

Explain what functions are and why they are useful.

Functions are a way of storing a process to be used over and over again. Functions can be passed and return variables.

Functions are useful because they reduce the amount of repetition needed, a set of instructions can be captured in a function and used by calling that function. They also make maintenance easier not only because it makes reading the code easier, it also means that rather than editing parts of the code in multiple places only the function need to be edited.
