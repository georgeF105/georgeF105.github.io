JavaScript Language

Parentheses ()

Parentheses are used encapsulate arguments for functions and to designate functions.

Brackets []

Square brackets are used to encapsulate array values and are used to hold the index value when accessing an array or an object.

Brackets {}

Curly brackets are used to contain the contents of a function or object and are used to contain the instruction of a control loop such as a for or while loop.

Quotes “” & ‘’

Double and single quotes can both be use to contain strings. One type can be used within the other to contain the quotes in the string e.g.: console.log(‘hi “hello”’) would print [hi “hello”] and console.log(“hi ‘hello’”) would print [hi ‘hello’].
