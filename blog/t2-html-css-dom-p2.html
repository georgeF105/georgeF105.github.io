Technical Blog Part 2

What is grid based design?

Grid based design is a way of laying out a web page that splits the page into 12 vertical divisions this gives the ability to split a page evenly into 2, 3, 4, 6 or even 12 columns or 1/3rd, 2/3rd and so on. This is an elegant way of vertically splitting up a page and simplifies CSS styling allowing styles to be reused though out the page.

Whats all the hype about responsive webpages?

A responsive webpage is one that will adjust to suit for different devices based on the width of the screen. This means that someone viewing a page will see a different simplified layout than someone viewing the same page on a tablet or laptop.
This make for a much easier to navigate and all-round better website.

What is semantic structure?

Semantic structure is the practice of using appropriate tags to define elements. An example would be using h1, h2, h3 tags etc. for headings / important text that needs to standout. These tags can be nested and makes styling the page a lot simpler.
