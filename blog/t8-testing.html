Scope

In its simplest form the scope of a variable describes where it can be used within a program.

A variable may have global or local scope.

Global variables can be accessed by everything within a webpage, they are declared outside a function. A variable is automatically global if it is declared without the var declaration in front of it.

Local variables can only be accessed from within the function that they were declared. Unlike some other programing languages variables declared within control loops, such as for and while loops, are not local to that loop and become part of the parent function scope.

Care must be taken when naming variables to not produce conflicts between global and local variables. When a conflict occurs the local variable takes precedence while within the function that the local variable was declared.

Local variable names can be repeated within different functions without worrying about conflict.
